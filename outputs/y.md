WHAT WE ARE SPECIFICALLY CHANGING:  CRL currently pays $29,455/month ($353,462/year) for HubStor Azure Blob Storage ($26,455/mo) and Azure Backup/DataBox ($3,000/mo) to store 1,041.7 TiB of scientific and regulatory data across 5 Azure regions. This project migrates all data to AWS S3 using AWS S3 File Gateway (EC2 t3.xlarge at each site presenting NFS/SMB shares to applications), terminates the HubStor subscription, and decommissions all Azure Blob storage. The change replaces a recurring managed SaaS storage platform with directly-managed cloud object storage at $4,496/month — a saving of $24,959/month from January 2027.
HOW SUCCESS WILL BE TRACKED AND MEASURED:  (1) MONTHLY COST REDUCTION — AWS bill confirmed below $4,500/month (vs current $29,455/mo) from January 2027 onwards, verified through AWS Cost Explorer and monthly finance reporting. (2) HUBSTOR INVOICE TO $0 — HubStor subscription invoice confirmed as $0 from January 2027, evidenced by contract termination confirmation and final billing statement. (3) PAYBACK ACHIEVED — Net cumulative saving turns positive within ~5 months of migration completion, tracked in the Financial Model Net Cumulative Saving column. (4) DATA INTEGRITY VERIFIED — 100% object count match between HubStor export and S3 inventory confirmed before each site cutover, documented in the migration runbook sign-off. (5) 4-YEAR BENEFIT — Projected net saving of ~$1.08M over 4 years (Jan 2027–Dec 2030) against the one-time migration investment of ~$121,000, representing an 84.7% cost reduction.
