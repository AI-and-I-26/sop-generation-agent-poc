WHAT WE ARE SPECIFICALLY CHANGING:
We are replacing HubStor (Cohesity) managed Azure Blob Storage with AWS S3 and AWS Storage Gateway across all six non-discovery sites. Specifically: the HubStor storage licence ($10,716/mo) and its associated Azure Blob accounts across four regions (EastUS2, UKSouth, WestEurope, WestUS2) are being decommissioned entirely, along with the proportional Azure Backup charge ($900/mo — 30% of the $3,000/mo total). In their place, all 312 TiB of non-discovery data will be migrated to AWS S3 Glacier Deep Archive and S3 Standard-IA in original file format via HubStor native export, AWS Snowball Edge, and AWS DataSync. Four AWS Storage Gateway File Gateway VMs (one per region) will replace every HubStor connector and agent at each site, presenting the S3 data as standard NFS/SMB shares — meaning sites access their files identically to before, but the underlying infrastructure is now AWS rather than HubStor/Azure. The migration begins with a three-month PoC on DEL-Microbial (June–August 2026), followed by 10–15 day cutover windows per site through October 2026, with full HubStor decommission and Azure Blob account closure by December 31, 2026. The one-time migration cost to execute all of this is $45,503 (including 10% contingency).

HOW SUCCESS WILL BE TRACKED AND MEASURED:
Success is measured across three dimensions. Financially: the monthly AWS bill (S3 + four Storage Gateway EC2s + CloudWatch/KMS/ops) must land at or below $1,128/mo from November 2026 onward, against the current $11,616/mo baseline — a $10,489/mo saving tracked by Finance each month against the Financial Model sheet in this workbook, with full payback of the one-time migration cost confirmed by April 2027 (4.3 months from decommission). Technically: each site cutover is only marked complete when three checks pass — S3 Inventory object count and total size match the NAS export manifest at 100%, a sample of files opened via Storage Gateway are confirmed readable in original format, and both an archive write test (file to S3) and restore test (file from S3) complete without error; Object Lock is additionally verified on all Safety Assessment GxP data before those connectors are switched. Operationally: success for each site is confirmed by the site lead in writing, and full project success is confirmed when all HubStor/Cohesity licences are cancelled with written decommission confirmation from the Cohesity Account Executive, all four Azure Blob account regions are closed with Azure-issued data erasure certificates, and the AWS monthly bill for December 2026 is reconciled against the $1,128/mo target with zero Azure or HubStor charges remaining.
